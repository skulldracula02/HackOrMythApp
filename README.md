# HackOrMythApp

## Purpose of the App

The purpose of this application is to test the user’s knowledge by allowing them to identify whether a statement is a "Hack" or a "Myth". The app is simple, interactive, and designed for beginner-level use.

---

## Features

* Start screen with a Play button
* Quiz screen with multiple questions
* Buttons for "Hack" and "Myth"
* Score tracking system (example: 3/4)
* Feedback after each question
* Review screen showing all questions and correct answers
* Restart button to try again

---

## Design Considerations

* The app uses an orange colour theme for a clean and modern look
* Large text sizes improve readability
* Buttons are placed side-by-side for easy interaction
* The layout is simple to avoid confusing the user
* The design approach is basic to match first-year level requirements

---

## How the App Works

1. The user opens the app and clicks "Play"
2. The quiz begins and questions are displayed one at a time
3. The user selects either "Hack" or "Myth"
4. Feedback is displayed after each answer
5. The user clicks "Next" to continue
6. The final score is displayed at the end
7. The user can review answers or restart the quiz
